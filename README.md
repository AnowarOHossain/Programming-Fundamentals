# Programming Fundamentals

A comprehensive collection of programming exercises and projects covering multiple languages to master core programming concepts and best practices.

## Languages Covered

- **C++** - Systems programming, object-oriented concepts, and performance optimization
- **Java** - Object-oriented programming, design patterns, and enterprise development
- **Python** - Scripting, data structures, and algorithmic problem-solving

## Repository Structure

Each language has its own directory containing numbered programs that progressively build upon fundamental concepts:

```
├── C++/        - C++ programs and exercises
├── Java/       - Java programs and exercises
├── Python/     - Python programs and exercises
└── README.md   - This file
```

## Purpose

This repository documents my learning journey through programming fundamentals, exploring different languages and their unique approaches to solving problems. Each program focuses on specific concepts like data structures, algorithms, control flow, and object-oriented design.

## Getting Started

Navigate to any language folder to explore the programs. Each program is self-contained and can be run independently.
